Project Objective

The goal of this project is to enhance a publicly accessible website by implementing real-world networking concepts. Traffic is proxied through Cloudflare (CDN/WAF) for optimization and DDoS protection. Additionally, the application features client-side XSS sanitization and Google Analytics for continuous network monitoring.
